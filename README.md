# 🏥 Diabetic Patients Readmission Analytics
This project predicts 30-day hospital readmissions using 100K+ diabetes patient [records](https://archive.ics.uci.edu/dataset/296/diabetes+130-us+hospitals+for+years+1999-2008) using Random Forest and Logistic Regression. I present the findings on Power BI.

## 💡 Purpose
Hospitals are able to identify high-risk patients before discharge by predicting 30-day hospital readmissions, which enables hospitals to intervene earlier, reduce preventable readmissions, and lower operational and penalty costs.
